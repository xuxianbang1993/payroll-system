# 导入导出需求

---

## 1. Excel 模板下载

- 提供标准 Excel 模板文件供用户下载填写
- 模板列：姓名、身份证号、公司简称、公司全称、部门、职位、人员类型、基本工资、补助、是否社保、是否地方养老、公积金金额

## 2. Excel 导入

- 支持 `.xlsx` 和 `.xls` 格式
- 导入时进行数据校验（姓名必填、金额为数字等）
- **冲突处理**：当导入数据中存在与已有员工同名+同身份证号的记录时，弹出冲突处理弹窗
  - 逐条显示冲突记录，用户可选择「覆盖」或「跳过」
  - 支持「全部覆盖」和「全部跳过」批量操作
  - 确认后执行导入

## 3. 导出功能矩阵

| 导出内容 | 格式 | 触发位置 |
|----------|------|----------|
| 员工花名册 | Excel (.xlsx) | 设置页 |
| 员工花名册 | CSV | 薪资核算页 |
| 全员薪资明细 | CSV | 薪资核算页 |
| 全员工资条 | PDF（批量） | 薪资核算页 |
| 凭证数据 | CSV | 凭证页 |

## 4. PDF 工资条

- 每人一页标准工资条
- 包含：组织名称、工资月份、员工姓名、收入明细、扣除明细、实发工资
- 支持批量导出为单个 PDF 文件（每页一人）
